Plugin adds custom fields WooCommerce products and calculate RRP based on it.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce_rrp` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
